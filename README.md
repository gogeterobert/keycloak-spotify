# keycloak-spotify

Keycloak Social Login extension for Spotify.

## Credits

Work based on [keycloak discord](https://github.com/wadahiro/keycloak-discord) and [an implementation of spotify IDP](https://github.com/JiriHartikka/keycloak/tree/spotify-social-idp).

## Install

Download `keycloak-spotify-<version>.jar` from [Releases page](https://github.com/gogeterobert/keycloak-spotify/releases).

## Source Build

Clone this repository and run `mvn package`.
You can see `keycloak-spotify-<version>.jar` under `target` directory.
